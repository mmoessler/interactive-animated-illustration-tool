Interactive and Animated
Illustration Tool

Motivation

An interactive and animated illustration of otherwise abstract concepts such as the law of large numbers (LLN) and the central limit theorem (CLT) in statistics.

The user can either set the inputs manually by using the sliders or by using the animation buttons which start an animation with an audio explanation of the results.

Examples

Example 1: Parameters of the Normal Distribution (Illustration I)
Example 2: Parameters of the Normal Distribution (Illustration II)
Example 3: Sampling distribution of the Sample Average

Approach

Step 1) Download the material

Download the material from the github repository

interactive-illustration-tool-main.zip

Unpack the .zip file and save the unpacked folder, interactive-illustration-tool-main, in a directory of your choice.

Add a subdirectory for your illustration, e.g., ./interactive-illustration-tool-main/illustration-01.

Step 2) Create figures

Create figure(s) for each parameterization, i.e., for each figure and each combination of parameter values.

Save all figures in a subdirectory using the following naming convention:

./interactive-illustration-tool-main/illustration-01/figures/figure_0X_a_b_c.svg,

where X is the index for the figures and a, b, c, ..., are the indices for slider 1, 2, 3, ..., respectively.

Step 3) Create HTML file

Choose a template according to the number of figures and sliders in your illustration:

Save the generated myHtml.html file in ./interactive-illustration-tool-main/illustration-01.

Step 4) Check the results

Check the results by opening the file ./interactive-illustration-tool-main/illustration-01/myHtml.html in your browser.

Step 4) Upload files to ILIAS

Compress all the material in the directory ./interactive-illustration-tool-main into a .zip file.

Upload the .zip file in a Lenrmodul HTML to ILIAS.

The illustration tool can be accessed by the following link:

https://ilias.uni-hohenheim.de/data/UHOH/lm_data/lm_1234567/interactive-illustration-tool-main/illustration-01/myHtml.html,

where 1234567 is the Object-ID generated by ILIAS which can be checked under the Info tag of the Lernmodule HTML.
